Colégio Estadual de Curitiba possui um grande projeto para melhora do meio ambiente.

Imagem do espaço Cultivando Saberes.

O projeto Cultivando Saberes é uma ideia nova e revolucionária com o intutito de estimular o jovem a interagir e aprender mais com o meio ambiente. O criador do projeto foi um professor de dentro da própria instituição estadual, conhecido pos todos como professor Gabriel Portugual Sorrentino de sociologia.

Para entendermos melhor desse projeto muito falado publicamente, nós fizemos algumas perguntas para o pioneiro do projeto.

Quais os objetivos do projeto?

Objetivos do Projeto:

  • Construir um espaço sustentável que possa promover comportamentos e valores que estão comprometidos com a sustentabilidade socioambiental, como: espaço físico na comunidade escolar.
  • Trazer conhecimento para os alunos voltas à questão socioambiental.

O que possui no espaço?

  • - Composteira
  • - Meliponário (abelhas nativas sem ferrão)
  • - Canteiros
  • - Sistema de captação de água da chuva
  • - Energia Solar
Imagem do Meliponário

Quais prêmios o projeto ja ganhou?

O projeto cultivando saberes foi vencedor nacional no desafio Escolas Sustentáveis, que foi organizado pelo Akatu e promovido pela ONU, que procura promover a sustentabilidade na Educação Básica, o projeto recebeu um financiamento de R$ 105 mil para desenvolver as ações do projeto
